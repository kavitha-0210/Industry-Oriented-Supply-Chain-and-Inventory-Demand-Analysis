# Industry-Oriented-Supply-Chain-and-Inventory-Demand-Analysis
This project analyzed inventory and demand data to support supply chain decision-making. Demand trends showed a steady increase throughout the year, indicating business growth and increasing customer demand.

Inventory turnover analysis helped evaluate inventory efficiency, while stock-out and overstock detection identified potential supply chain risks. A Linear Regression model was used to forecast future demand, enabling proactive inventory planning.

The results demonstrate how data analytics can improve inventory management, reduce holding costs, prevent stock shortages, and support efficient supply chain operations.
